Advance Each - Simple loop with custom format

    {{#:each {@$dataStore}members }}
  • {{:fn '{$}getTitle' gender first_name }} {{ first_name }}
    • {{/:each}}

Advance Each - Alias

    {{#:each {@$dataStore}members alias='member,index' }}
  • {{ index }} - {{ member.first_name }}
    • {{/:each}}

Advance Each - accessing array index using @index

    {{#:each {@$dataStore}members }}
  • {{ @index }} - {{ first_name }}
    • {{/:each}}

Advance Each - sort by First Name

    {{#:each {@$dataStore}members sortOn='first_name' sortBy='asc' }}
  • {{ first_name }} {{ last_name }}
    • {{/:each}}

Advance Each - filter using function

    {{#:each {@$dataStore}members fnFilter='{$}aFilter' }}
  • {{ first_name }} {{ last_name }}
    • {{^}}
  • No Records.
    • {{/:each}}
